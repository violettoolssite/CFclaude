# 贡献指南

感谢您对 CFclaude Code 项目的关注！

## 如何贡献

### 报告问题

1. 在提交新问题前，请先搜索现有的 Issues
2. 使用清晰的标题描述问题
3. 提供复现步骤、预期行为和实际行为
4. 附上相关的日志或截图

### 提交代码

1. Fork 本仓库
2. 创建功能分支: `git checkout -b feature/your-feature`
3. 提交更改: `git commit -m 'Add some feature'`
4. 推送到分支: `git push origin feature/your-feature`
5. 提交 Pull Request

### 代码规范

- 使用 TypeScript 编写代码
- 遵循现有的代码风格
- 添加必要的注释和文档
- 确保所有测试通过

### 开发环境

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 行为准则

请保持友善和尊重。我们欢迎所有形式的贡献，无论大小。

## 许可

您提交的贡献将在 Apache 2.0 许可证下发布。
