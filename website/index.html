使用指南

三步开始使用 Claude Code

1

安装 Node.js 和 Claude Code

下载并安装 Node.js 18+，然后运行以下命令安装 Claude Code：

npm install -g @anthropic-ai/claude-code
2

获取 API Key

前往你选择的服务商平台注册并获取 API Key：

DeepSeek Kimi 豆包 通义千问
3

配置并使用

打开 CFclaude，选择服务商，输入 API Key，点击"应用配置"。然后在终端中运行：

claude
